    Kodama 程序 [app][edit]

      Kodama 是一款用于构建双链风格笔记的命令行程序. 是笔者为自己打造的博客工具、笔记软件、维基系统. Kodama 程序 所生成网站的默认风格受到软件 Forester 的深刻影响.

      受限于 Markdown 本身表达能力的缺陷, Kodama 无法像 Forester 那样完全控制文档数据的处理模式. 在其他方面提供了更多便捷的特性, 包括但不限于:

      色彩特性 [feature-color][edit]
        • 内置一套精心设计的暗主题, 明暗主题切换默认遵循浏览器主题偏好.
        • 可以安装其他色彩风格的主题, 主题文件仓库位于 Github - themes.
        • 使用 Typst 所生成的公式或图像中的文本或线条颜色, 会自动的跟随颜色主题方案而变化, 无论在何种主题之下都有正确清晰的显示效果.
        • 用户可以同时为网站部署多种主题, 并灵活地在他们之间切换. 即使页面刷新或者浏览器关闭, 之前所选的主题方案也不会受到影响.

          多语言文本优化 [feature-text][edit]

            这个礼拜我们聚集了全国数学精英做 2025 annual math review. 讲他们的重要工作。我本人会參加每一埸演讲。但是我很失望：求真書院的学生參加的学生很少，都躲在甚么地方了？做世界一流的研究，就如攀登高峰，需要長期準備工作，準備好工具。当年衛青 霍去病出塞征伐匈奴，入千里無人之地，义無反顧，弱则死于绝域，霍去病年不过二十，八百骑直走匈奴王庭，第一次成功的擊败匈奴，抵定河西走廊，立中国千載之功！何其壮哉！今我求真子弟，不思科研之極至，立不朽之功。躲在家中玩耍游戏，置一流学問於不顾！本人六十年来，日出而作，每天工作至少八个小时，猶恐不及，未達天人之交，有负家国之望！竊为诸君今日所志所为，羞之羞之！

            节 [section][edit]

              节 是 Kodama 写作 中的基本概念. 每个 节 都将被处理为一个 HTML 文件. 节 可以来自 Markdown 文档或 Typst 文档. 节 之间的 链接 和 组合 构成了 Kodama 写作 的基本模式.
